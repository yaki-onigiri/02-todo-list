# TODOリスト

ーーーーー

## 概要

JavaScriptで作成したシンプルなTODO管理アプリ

ーーーーー

## 主な機能

・タスク追加
・タスク削除
・Enterキーで追加
・「チェックボックス」でタスクを完了したことをわかりやすく表示
・個別で削除できる
・全削除
・完了タスク一括削除
・localStorage保存
・未完了タスクフィルター
・タスク編集機能（ダブルクリック）
・フィルター状態の保存

ーーーーー

## 使用技術

HTML
